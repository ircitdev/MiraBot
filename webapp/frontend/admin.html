info Процесс онбординга

Текущий процесс (v2.0 - Phase 2 Complete)

Касание 1: Приветствие и запрос имени

  • Приветствие Миры с паузами между сообщениями
  • Запрос имени пользователя
  • Информация о конфиденциальности
  • Выбор "боли" (6 кнопок-вариантов)

Касание 2: Социальный портрет

  • Свободный ответ: возраст, город, работа
  • Данные сохраняются для AI-парсинга

Касание 3: Семейный статус

  • Выбор статуса: Замужем / В отношениях / Свободна / Всё сложно
  • Опция "Расскажу позже"

Касание 4: Детали о семье

  • Запрашивается только для замужних/в отношениях
  • Свободный ответ: имя партнёра, наличие детей
  • Опция "Расскажу позже"

Завершение:

  • Эмпатичный отклик: "Теперь я вижу тебя 💛"
  • Подарок: 3 дня Premium без ограничений
📚 Документация: См. docs/onboarding/ONBOARDING_TESTING_GUIDE.md и docs/onboarding/TODO_ONBOARDING.md
